Abstract

The invention belongs to the technical field of computer information security, and especially relates to a network informatization security application device. The network informatization security application device is characterized by comprising a computer network control intelligent module, a network information receiving module, an information security intelligent identification module, an information intelligent processing module, a network information intelligent contactor, a computer hard disk power supply, an intelligent hard disk, an intelligent microprocessor, an interface conversion portion and a computer serial port. According to the invention, computer information can be well protected when the computer is connected to the internet, the computer hard disk with information in the user computer is effectively protected, and therefore hackers cannot steal computer hard disk data and attack the computer hard disk.
