Abstract

A home network system and a method for processing events therein are provided to minimize operation loads and power consumption by classifying events according to event generation modes and applying different transmission rules according to the natures of generated events. If a state change occurs in a slave device(S100), the slave device judges whether the present state is an event generation mode or an event generation adjustment mode(S110). If an information event is generated in the event generation adjustment mode, the slave device judges whether it is set to transmit an associated event when an event situation occurs. In case the slave device is set to transmit an associated event, the slave device creates an event file and transmits it to a master device through a bus network(S120). If a warning event is generated in the event generation adjustment mode, the slave device judges whether it is time to generate an associated event. If it is time to generate an associated event, the slave device periodically creates and transmits an event file(S130). If an error event is generated, the slave device immediately creates an event file and transmits it to the master device through the bus network(S140).
